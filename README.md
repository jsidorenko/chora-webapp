# choratech-webapp

choratech client application

## Technology Stack

- React 16.2.0
- Redux 3.7.2
- Webpack 3.11.0
- Solidity 0.4.19
- Truffle 4.0.6

## Development Environment

- run `truffle develop`
- in the truffle console, run `compile`
- in the truffle console, run `migrate`
- open a new tab, run `yarn start`
